# Graph-Bridge-v2
AI-powered Blueprint graph assistant for Unreal Engine. Chat with Claude or any GPT inside the editor to inspect, summarize, and manipulate your graphs.


## Installation
1. Download GraphBridgev2_v1.0.0_UE5.7.zip
2. Extract into YourProject/Plugins/GraphBridgev2/
3. Open your project — when prompted, enable the plugin and restart
4. Go to Window → GraphBridge AI


## First Use
1. Click Start Server — green dot confirms it's running on port 8080
2. Enter your Anthropic API key (sk-ant-...) and click Save Settings
3. Open any Blueprint in the editor
4. Click List All Nodes or type a question and hit Run Task
